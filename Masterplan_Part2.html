3D Printed Custom Robot Gripper + Tactile Sensing

Design, print, and deploy a custom smart gripper that can feel what it's holding.

🎯 Core Objectives

  • Design a custom compliant gripper in FreeCAD — TPU flex fingers for soft grasping. Print on Kobra 2 Neo.
  • Add FSR (Force Sensitive Resistor) sensors to gripper fingers — wire to Arduino Nano for analog read
  • ROS2 node reads force data from Arduino (serial) and publishes to /gripper/force topic
  • Implement grasp control: close until force threshold met → maintain grip → release on command
  • Demo: robot grasps an egg (or paper cup) without crushing it — tactile-guided grasping
  • Benchmark: 20 fragile object grasps. Compare force-controlled vs. position-controlled success rates.
  • Document full design files on GitHub (FreeCAD source + STL + Arduino code + ROS2 node)

🛠️ Tools & Tech

  • FreeCAD (gripper design), Anycubic Kobra 2 Neo (TPU filament), Arduino Nano, pyserial
  • FSR402 force sensors (~₹200 each), ROS2 serial bridge, custom URDF for new gripper
  • Matplotlib for force-vs-time plots, statistical comparison of both control methods
